# 🍔 Sabor & Ponto – Controle de Vendas

Aplicação web desenvolvida para auxiliar no controle de vendas de uma lanchonete fictícia chamada **Sabor & Ponto**.
O sistema permite registrar produtos vendidos, calcular automaticamente o total das vendas e armazenar os dados localmente no navegador.

Este projeto foi desenvolvido como parte da atividade de **Projeto de Extensão** do curso de **Análise e Desenvolvimento de Sistemas**.

---

## 🚀 Tecnologias utilizadas

* **React**
* **JavaScript**
* **Vite**
* **HTML5**
* **CSS3**
* **LocalStorage** para armazenamento dos dados

---

## 📋 Funcionalidades

✔ Cadastro de vendas
✔ Cálculo automático do total das vendas
✔ Listagem das vendas registradas
✔ Armazenamento local dos dados no navegador
✔ Interface simples e intuitiva

---

## 📱 Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/RuanQuadros/Projeto-Wyden.git
```

2. Acesse a pasta do projeto:

```
cd Projeto-Wyden
```

3. Instale as dependências:

```
npm install
```

4. Execute o projeto:

```
npm run dev
```

5. Abra no navegador:

```
http://localhost:5173
```

---

## 🎯 Objetivo do Projeto

O objetivo do projeto é oferecer uma solução simples para pequenos estabelecimentos que realizam o controle de vendas manualmente, permitindo registrar vendas digitalmente e visualizar o faturamento diário.
